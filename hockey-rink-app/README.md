# Hockey Rink App

## Overview
The Hockey Rink App is a web application that simulates an ice hockey rink. Users can click on the rink to "shoot" at various locations, and the application will display the shot location.

## Features
- Interactive ice hockey rink
- Clickable areas to simulate shooting
- Dynamic display of shot locations

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd hockey-rink-app
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage
1. Start the server:
   ```
   npm start
   ```
2. Open your web browser and go to `http://localhost:3000` to view the application.
